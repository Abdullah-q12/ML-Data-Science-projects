# Task 1: Data Analysis & Visualization

## Project Goal
Perform exploratory data analysis (EDA) and visualization on datasets like Titanic or Iris to gain insights and identify patterns.

## Objectives
- Load dataset into Pandas DataFrame
- Inspect data types and summary statistics
- Identify missing values
- Count unique values in categorical features
- Visualize categorical and numerical variables
- Plot correlation heatmap for numerical features

## Tools & Libraries
- Python 3.x
- Pandas, NumPy
- Matplotlib, Seaborn

## Outcome
- Clean and well-structured visualizations
- Insights about dataset distributions, relationships, and patterns
- Reproducible analysis pipeline
