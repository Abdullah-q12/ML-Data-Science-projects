# Task 2: Titanic Preprocessing & Feature Engineering

## Project Goal
Prepare the Titanic dataset for machine learning by handling missing values, encoding categorical features, scaling numerical features, and creating new features.

## Objectives
- Impute missing values (median for numerical, mode for categorical)
- Encode categorical variables (One-Hot Encoding / Label Encoding)
- Scale numerical features using MinMaxScaler or StandardScaler
- Create new features:
  - `Title` extracted from passenger names
  - `FamilySize` combining SibSp and Parch

## Tools & Libraries
- Python 3.x
- Pandas, NumPy
- Scikit-learn

## Outcome
- Preprocessed dataset ready for modeling
- Features standardized and categorical variables encoded
- Additional features enhancing predictive power
