# Task 5: Hyperparameter Optimization

## Project Goal
Optimize machine learning models for Titanic survival prediction by tuning hyperparameters.

## Objectives
- Select models (e.g., Logistic Regression, Decision Tree)
- Define hyperparameter grids
- Use GridSearchCV or RandomizedSearchCV
- Evaluate optimized models on test set
- Compare baseline vs optimized model performance

## Tools & Libraries
- Python 3.x
- Scikit-learn

## Outcome
- Best hyperparameters identified
- Improved model performance
- Clear comparison with baseline
